Stay Sharp

Tuner & Metronome for Guitar, Bass, Ukulele & More

Stay Sharp is the all-in-one practice tool built for guitarists and musicians of all levels. Tune accurately, keep perfect time, and build better practice habits — all in a clean, fast app.

Tuner

  • Accurate pitch detection for guitar, bass, ukulele, banjo, mandolin, and violin
  • Animated needle meter with real-time cents display
  • String-by-string guided tuning mode — auto-advances when each string is in tune
  • Capo support (frets 1–7) for guitar and 7-string
  • Drone tone — sustained reference pitch for tuning by ear
  • Auto mode detects which string you're playing
  • Alternate tunings: Drop D, Open G, Open D, DADGAD, Open E, Drop C, and more

Metronome

  • 10 time signatures: 2/4, 3/4, 4/4, 5/4, 6/4, 7/4, 6/8, 7/8, 9/8, 12/8
  • Smart default accents for compound meters (6/8, 9/8, 12/8)
  • Subdivisions: quarter, 8th note, triplet, 16th note
  • Custom accent patterns — tap any beat to toggle accent
  • Tap tempo + BPM nudge buttons (±1, ±5)
  • 20–300 BPM range
